# 机场频率查询系统

一个现代化、响应式的机场频率查询工具，允许用户通过机场名称或机场代码快速查询各类通信频率信息。

## 功能特性

- 支持通过机场名称或四字代码查询频率信息
- 提供塔台、地面、进近、放行和区域管制五种频率查询
- 快速访问常用机场功能
- 一键复制所有频率信息
- 响应式设计，支持各种设备访问
- 现代化UI，流畅的动画效果
- 错误处理和用户友好提示

## 技术栈

- HTML5
- CSS3 (使用Tailwind CSS v3)
- JavaScript (原生)
- Font Awesome 图标库

## 使用方法

1. 在搜索框中输入机场名称（如：北京首都）或机场代码（如：ZBAA）
2. 点击"查询"按钮或按回车键
3. 查看查询结果中的各类频率信息
4. 点击"复制所有频率"按钮可一键复制完整频率信息
5. 也可点击快速访问区域的常用机场代码进行快速查询

## 频率类型说明

- **塔台频率**：用于起飞和降落阶段与塔台管制员通信
- **地面频率**：用于机场地面移动时与地面管制员通信
- **进近频率**：用于接近机场和离场阶段与进近管制员通信
- **放行频率**：用于获取放行许可
- **区域管制频率**：用于航路飞行时与区域管制员通信
